# SmartFine

SmartFine is a Traffic Fine Management System. 

With this project, you can:
- **Issue Fines**: Allow traffic officers to issue fines digitally using the mobile application.
- **Pay Fines**: Allow citizens to make payments online for their traffic violations.
- **Manage Services**: Manage traffic fine records, notification dispatches (via SMS/email), auth policies, and view generated reports through microservices.
